# ⚽ Football Database & Performance Analysis

## 📌 Overview
This project simulates a football club database system used to track players, matches, and performance statistics.

## 🎯 Objective
To analyze player and team performance using SQL queries.

## 🛠️ Tools Used
- MySQL Workbench
- SQL
- GitHub

## 📊 Features
- Track goals and assists per player
- Analyze team performance
- Identify top-performing players
- Calculate player consistency

## 🔥 Key Insights
- Top goal scorer: Akia
- Best playmaker: Bongz
- Most consistent player: Rele

## 🧠 Skills Demonstrated
- SQL Joins
- Aggregations (SUM, AVG)
- Database Design
- Real-world data analysis

## How to Use
1. Create the database in MySQL
2. Run the schema file
3. Insert sample data
4. Run queries from queries.sql

## Future Improvements
- Add more tables(teams, matches)

- Integration with Python or Dashboards
